The Journey of N00b : Roadmap

post @ 2020-06-30
/2020/06/30/The-Journey-of-N00b-Roadmap/roadmap.png

GAME HACKING IS NOT FOR LAZY, STUPID OR IMMATURE PEOPLE!!

What skills do you have to:
+Intellligence
+Hard Working
+Learning by yourself
+Lots of time and praticing

The more you practice, the more you learn

Basic Timeline for Learning:

1 Week : Learn The absolute basic of using Cheat Engine(This Post)
2-3 Week : Learn enough CE to use all it’s features, how to find pointers etc..
1 month : Learn C++/C# well enough to make a hello world and know the absolute basics of coding
3-6 months : Experienced enough to make very basic trainers in C++/C#
6-12 months : Learn enough to make trainers with more advanced feature like patern scanning,teleporting etc…
1-2 Years : Intermediate coding/reversing skills to hack any game you want without anticheat
2 Years : Start reversing anticheat
